# mybuildboard

MyBuildBoard is an AI-powered developer portfolio board that helps developers showcase:

- Projects
- GitHub repositories
- Technical achievements
- Professional experience

## Run locally

Open `/home/runner/work/mybuildboard/mybuildboard/index.html` in a browser.

## What is included

- Customizable interactive board with drag-and-drop cards
- Card types for project, repository, achievement, and experience
- Local persistence using `localStorage`
- AI spotlight generator that creates a concise portfolio summary from board content
